# /caveman-commit

Herschrijft een git commit-bericht naar holenman-stijl.

**Gebruik:**  
`/caveman-commit "fix: out-of-bounds bij lege array in renderfunctie"`

**Resultaat (speer):**  
`fix oob lege array render`

**Regels:**

- Max 50 tekens
- Alleen essentiële woorden
- Type prefix (`fix`, `feat`, `chore`) blijft behouden
